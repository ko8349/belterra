# Belterra Residences — Final GitHub Production Pack

Production-ready React/Vite website for **Belterra Residences by Aventus**.

## What is included

- Approved Belterra logo
- Approved project renders in `src/assets/renders`
- Original architectural layout screenshots stored in `src/assets/source-layouts`
- Brand-aligned masterplan section based on the real 70-unit / B1–B8 structure
- Brand-aligned typology cards based on the real house type mix
- Lead CTAs for price list and WhatsApp
- Responsive mobile layout
- SEO meta tags
- Vercel/GitHub-ready setup

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

The production build will be created in the `dist/` folder.

## Deploy to Vercel

1. Upload this folder to GitHub.
2. In Vercel, import the repository.
3. Use:
   - Framework: Vite
   - Build command: `npm run build`
   - Output directory: `dist`

## Connect GoDaddy domain

1. Deploy on Vercel.
2. Add the domain in Vercel.
3. Copy Vercel DNS records into GoDaddy DNS settings.
