## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Create a GitHub repository.
2. Upload all files from this folder.
3. Go to Vercel.
4. Import the GitHub repository.
5. Use:
   - Framework: Vite
   - Build command: `npm run build`
   - Output directory: `dist`
