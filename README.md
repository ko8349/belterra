# Belterra Web v2 — Clean Brand + Redrawn Layouts

This version fixes the messy logo and raw layout screenshots.

## What changed
- The navbar no longer uses the tiny full-logo image. It uses a clean CSS Belterra wordmark/monogram.
- The approved logo image is still included as a reference asset.
- The masterplan is redrawn as a brand-aligned schematic using the real B1–B8 / 01–70 structure.
- House types are redrawn as clean buyer-facing diagrams based on the real typology package.
- Original architectural screenshots remain included in a collapsible source-reference section.

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy
Use Vercel:
- Framework: Vite
- Build command: `npm run build`
- Output directory: `dist`
